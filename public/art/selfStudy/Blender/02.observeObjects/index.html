02. 观察对象 #

  • 以对象为中心旋转
    • 按住 鼠标中键 不放并同时进行拖拽，这时观察者视角将会以对象的「原点」(在这里暂且称为「原点」，因为这个点用户可以根据自己的需要而改变位置)
    • 当打开了 模拟3键鼠标 设置后，可以以「按住 Alt 不放并对画面进行拖拽」代替此操作
    • 在 漫游器 上点击 鼠标左键 不放并同时进行拖拽

正交视图 / 透视图

切换当前视图为正交视图 / 透视图 按钮
  • 观察角度快速转换
    • 同时打开四视图: Ctrl + Alt + Q
    • 正交顶视图: 小键盘数字键 7
      若要进入正好相反的 正交底视图 : Ctrl + 7，下同
    • 正交前视图: 小键盘数字键 1
    • 正交右视图: 小键盘数字键 3
    • 相反视图: 小键盘数字键9
      在正交视图使用此快捷键，可以快速将视图移动到选中对象对侧对称的位置
      而在透视图状态下，使用此快捷键，则视图只会移动到以 Z 轴 对称的位置，而不会让视图上下颠倒
    • 快速聚焦到单个对象: 小键盘小数点 . (此键不可以被字母区的句点代替)
      • 如果在四视图模式下，用 Ctrl + . 可以同时把选中的对象在四个不同视图中聚焦
    • 仅显示选中对象: 正斜杠(小键盘的÷号) / (同时影响四视图，某种程度上可以替换上一个快捷键)
    • 回到场景的默认观察视角: Home

摄像机视角的补充 #

进入摄像机视角后，如果你此时再对视角进行调整，你会发现，摄像机并没有随着你的视角调整而改变位置

如果你需要摄像机随着你的视图调整而移动位置，你可以执行以下的操作，以便让你的摄像机随着视图调整而移动位置:

  1. 选择场景中的某个摄像机
  2. 按下N键以打开 N 键面板
  3. 勾选位于 视图锁定 > 锁定 下的 锁定摄像机到视图方位

这时候你也许会问，打开锁定摄像机到视图方位后，为什么会有一个框在视图中间？这个框又代表什么呢？

别急，且听我一一道来

首先，这个框代表的是你作品的 「最终成品预览区域」 ，也就是说如果你需要渲染这台摄像机看到的景象的时候，渲染的结果只会包含这个框中看到的区域

其次，这个框的大小，你是可以根据需要自行修改的，修改方法如下:

  1. 从场景中选择需要改动分辨率的摄像机
  2. 从 Blender 右侧的参数面板中找到 输出属性 > 格式 > 分辨率 X / Y
  3. 根据自己的需求，修改对应的参数

这时，你也许还会问，我已经调整好了摄像机的分辨率，但我觉得这个可视范围框框有点碍眼，而且在视图中的位置也不是我想要的位置，我该怎么放大缩小？又该怎么移动它的位置呢？

要达到这样的效果，我们只需要重新打开 N 键面板 关闭 锁定摄像机到视图方位

然后就可以用滚轮来放大缩小该区域，或者通过按住 Shift + 鼠标中键 不放，同时移动鼠标

这时，你就会发现，这台摄像机的可视范围会跟随你的鼠标进行移动啦

最后我们再重新打开 锁定摄像机到视图方位 就可以在一个比较舒服的视图下调成摄像机的位置了
