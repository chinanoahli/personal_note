01. 下载与偏好设置 #

如何下载 #

下载 Blender 可以到其官网，也可以直接到国内的阿里云镜像站下载

下载完成后，运行安装程序，并按照提示安装即可

偏好设置 #

以下的笔记，基本上是以 Blender 自带的 简体中文 翻译版本的作为对象记录，必要时会同时提及该项目的英文原名

第一次打开 Blender 时，默认为英文UI，我们首先需要在顶部菜单 Edit > Preferences… 打开 Blender 偏好设置 (Blender Preferences) 窗口

界面 (Interface) #

然后在左侧的窗格中找到 界面 (Interface)，并在右边的窗格中找到 翻译 (Translation) > 语言 (Language)，把它从**English (English)** 改为 Simplified Chinese (简体中文)

假若你在高分屏中使用 Blender，你可能会觉得 Blender 的UI及字体很小，你可以把 显示 > 分辨率缩放 调整得大一点，此值默认为 1.0

在 Blender 每次启动时，都会自动显示 启动画面，你若不需要启动画面，可以同时将 显示 > 启动画面 取消勾选

在 翻译 中，有一个名为 新建数据 的选项，它会导致某些插件出错，所以这里它关掉

在 文本渲染 设置里，我们可以根据自己的喜好，选择 Blender 的 界面字体 以及部分功能需要用到的 等宽字体，同时 抗锯齿 选项可以根据自身的需要决定是否打开

在 编辑器 > 状态栏 下，可以找到常用的系统软硬件状态数据，如 系统内存 和 显存，勾选后 Blender 将会在主窗口的右下角显示这些数据

输入 #

如果你的键盘没有小键盘区域，你可以导航到 键盘 并勾选 模拟数字键盘 选项，这个选项可以让 Blender 将键盘字母区上面一行数字键当作是小键盘快捷键

如果你曾经用过别的3D建模软件，你还可能需要开启 鼠标 > 模拟3键鼠标 项，以便可以使用 Alt + 拖拉鼠标 来改变观察角度

视图切换 #

Blender 默认的观察角度操作方式是以 「观察者视角的焦点」 作为 视图旋转的原点 ，你可以在 旋转 & 平移 中勾选 围绕选择物体旋转，这样 Blender 在旋转观察者视角时，就会以当前选中的物体作为旋转中心

如果不勾选 旋转 & 平移 > 自动 > 深度，那么你在推拉观察者视角的镜头时将会出现极限，即镜头推拉到一定程度之后就不能继续推拉

系统 #

如果你的硬件有独立显卡，你希望使用独立显卡进行辅助渲染(CUDA)，又或者想要渲染光线追踪(OptiX)效果，那么你就需要在 Cycles 渲染设备 中分别在 CUDA 和 OptiX 标签页中分别勾选你的独立显卡

⚠ 注意：

建议你在设置完成后，建议把 CUDA 选项卡点选为激活状态

在我的环境 (Blender v3.5.1) 中，如果不把 CUDA 选项卡激活，那么在渲染时，Blender 将无法正确调用独立显卡进行渲染，所有渲染任务将由 CPU 承担

建议勾选 内存 & 限额 中的 全局撤销 ，因为只有勾选了这个选项，Blender 才能使用更精细的撤销控制

保存 & 加载 #

在 Blend 文件 分类下，有一个名为 自动运行 Python 脚本 的选项。部分 Blender 项目会包含一些 Python 脚本，如果勾选此项目，则在打开这种项目时，它所包含的 Python 脚本会被自动运行。此项目需要你根据自己的安全准则决定是否打开

建议勾选**Blend 文件** 下的 自动保存 选项，以防因工程占用太多系统资源，造成卡顿时，因为没有及时保存而前功尽弃

插件 #

推荐打开下列 Blender 自带的插件 (若找不到，可以用搜索框快速定位)

  • 节点: Node Wrangler
  • 绑定: Rigify
  • 导入-导出: 导入图像为平面
  • 导入-导出: Export Autodesk DXF Format (.dxf)
  • 导入-导出: Import Autodesk DXF Format (.dxf)
  • 添加曲线: Extra Objects
  • 添加网格: BoltFactory
  • 添加网格: Extra Objects
  • 添加网格: A.N.T.Landscape
  • 界面: Copy Attributes Menu
  • 界面: Modifier Tools
  • 网格: LoopTools
  • 物体: Bool Tool
  • UV: Magic UV

保存设置 #

尽管在 Blender 3.5.1 中，默认开启了 自动保存设置，但在每次更改偏好设置后，手动找到在 Blender 偏好设置 窗口左下角的 ≡ 按钮进行一次手动的 保存用户设置 仍是一个好习惯
