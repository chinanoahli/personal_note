04. 场景单位、对象的选择与添加对象 #

场景单位 #

在上一节 变换和移动对象 中提到，Blender 的部分组合键是可以嵌入数值来实现位移和变换对象

而这个数值的单位默认是 「米」，那有没有办法可以把这个「米」修改成别的单位呢？

答案是肯定的，我们只需要在 Blender 窗口右侧的属性面板中找到 场景属性 面板，然后根据需要修改其中的 单位 > 单位系统 即可

对象的选择 #

与其他任何编辑类软件类似，Blender 也提供了非常人性化的 单选 、 多选 等选择对象的操作

通过 导航器 选择对象 #

在这里，先来说说 导航器 中选择对象的操作

导航器的操作相对直观，它与很多其他软件类似，提供了两种操作方式:

  1. 通过 Shift 进行快速选择在导航器中连续排列的对象

    1. 选择第一个需要选择的对象
    2. 按住 Shift 不放
    3. 选择最后一个需要选中的对象

  2. 通过 Ctrl 实现跳跃式多选

    1. 按住 Ctrl 不放
    2. 逐一点击需要选中的对象

  3. 取消选择某一个对象
    有时候我们需要从已经选择的多个工具中取消一些选中的对象，这时我们可以执行下列的操作

  4. 按住 Ctrl 不放

  5. 逐一点击需要取消选中的对象
    ⚠ 有时需要多点击一次才能取消选择，被选择的对象会被标记成 蓝色高亮

从 3D 视图中选择对象 #

通过 Shift 进行选择 #

在 3D 视图中，我们可以通过按住 Shift 不放，逐一点击场景中需要选择的多个对象来进行多选，取消选择也是同理，只需要按住 Shift 不放，再次点击已经被选中的对象即可取消选择

通过 框选 工具进行选择 #

鼠标左键长按 调整 工具不放，待工具子菜单弹出来后，把光标移动到 框选 工具上再放开鼠标左键，即可进入 框选 工具

你也可以通过键盘的 W 键来进行快速切换(包括后面提到的另外两种选择工具都可以切换到喔)

⚠ 在进入框选工具后鼠标将暂时无法再进行视图旋转的操作

框选工具有5种模式:

  1. 开始新的选择
    每次拖出选择框都不会保留之前已选中的项目
  2. 扩展当前选中项
    保留已选中的项目，并加选现在框选到的项目
  3. 从当前选中项中减除
    从已选中的项目中移除框选到的对象 (不会影响未被选择的对象)
  4. 反转所有选中项
    被框选到的对象反转选中状态 (即选中的会被取消选中，未选中的则会被选中)
  5. 与现有选中项相交
    仅保留选中在多次框选中都被选中的对象

通过 刷选 工具进行选择 #

同样地，刷选 也拥有 开始新选择 、 扩展当前选中项 以及 从当前选中项中减除 这3种不同的选择模式

与框选工具不同的是，刷选 工具会让你的光标暂时变成笔刷，笔刷的大小可以被调整，在复杂场景中需要连续选择一片对象时会非常有用

从下面的GIF中你可以看到调整 半径 后，鼠标光标将会随着半径的大小而改变大小

通过 套索选择 工具进行选择 #

与任何平面图像设计软件(Photoshop、Clip Studio和SAI等等)类似，Blender 也提供了 套索选择 工具，且其用法基本相似

套索选择工具允许你通过鼠标光标画出任意形状蚂蚁线，并根据选择模式对蚂蚁线圈中的对象进行加选或减除

套索选择工具很好地避免了使用框选工具时，必须使用矩形框而导致经常选择到不需要的对象的尴尬

同样地，套索选择工具拥有与框选工具同样的5中选择模式

在 3D 场景中的选择操作键盘快捷键 #

在 3D 视图中，要全选场景里面的对象，我们只需要按下键盘的 A 键即可

要取消全部选中对象的话，只需要连续按下 两次 的 A 键即可

当然了，Blender 默认还提供了令一组快捷键，即 Alt + A 来进行取消全部选择，但是以后会介快捷操作，这个快捷操作将会占用 Alt + A 这组快捷键，所以这里并不推荐使用这组快捷键进行取消选择

添加对象 #

要在 Blender 场景中添加对象、灯光以及摄像机等等，我们不仅可以通过位于顶部工具栏的 添加菜单 来添加对象，还可以通过快捷键 Shift + A 来快速打开添加菜单

删除对象 #

要在 Blender 场景中删除对象，即可以使用大部分 3D 软件都通用，键盘上的 Delete 键，也可以使用 Blender 独有的 X 键

在按下 Blender 独有的 X 键后，软件会在你光标的附近弹出一个确认框，此时用鼠标点下确认框中的 删除 按钮即可

这里只需要根据个人习惯进行选择即可，如果你习惯以右手使用鼠标，那么则推荐使用 X 键，因为它位于键盘的左手测，可以避免你经常移动左手(或为了用右手按下Del键而不得不放开鼠标)，以便提高工作效率

附录: 体验产品创作过程 #

附录提到的文件，请到原视频教程 BV1zh411Y7LX 的简介信息中下载

先将 Blender 默认工程中的立方体删除掉

然后从 Blender 的菜单栏中找到 文件 > 关联

并在弹出的文件浏览器中找到 辣椒教材汇总 > 黑铁骑士Ⅱ配套文件 > 第零章(摩托在此~) > 02基础入门与动画案例 > 素材.zip(需提前解压出来) > 摩托.blend > Collection > 摩托

然后点击文件浏览器右下角的 关联 按钮

若你的电脑性能可以，可以在 3D 视图右上角，把 视图着色方式 调整成 「渲染」，这样你就能直接在 3D 视图上大致看到追踪渲染的结果

如果电脑性能不太行则不建议开启这个功能

此时，你应该能看到一台摩托车出现在场景地平面的原点上，如果你觉得摩托车有点小，则可以根据自己的需要进行缩放

然后移除我们用不到的自带点光源，并新建一个 面光，并把面光移动到摩托车的右上方，光照方向朝向摩托车(面光上会有一条垂直线，这就是光照的方向，拖动垂直线上的小原点即可改变光照方向)

默认情况下，灯光非常暗，这时候我们就需要在属性面板中找到面光的 物体数据属性 并把能量调到合适的大小

这时我们重复上面创建灯光的操作，以摩托车的位置为对称轴，在摩托车的另一边新建一个同样的面光，然后在 Y 轴方向上，把两个面光的长度拉伸到足以能照亮整台摩托车

目前的画面还没有足够的看头，因为它没有任何的背景环境，我们需要通过下面的操作导入一个背景

先在 Blender 的右侧面板中找到 世界属性，然后点击 表(曲)面 > 颜色 前面的 小黄点，再弹出的菜单中选择 环境纹理

此时整个场景都会变成紫色，这时因为我们还没有导入背景图像

接下来，我们继续点击位于 颜色 下一行的 打开 按钮，并从之前的附件位置中，找到 金丝雀码头_hdr.hdr，然后点击文件浏览器右下角的 打开图像 按钮即可

接下来，我们还需要挑选一个拍摄角度，活用第二篇笔记中的技巧，挑选一个不错的角度，然后按下 Ctrl + Alt + 0 把它设置成摄像机视角

接下来是特效的微调(后面的课程会详细讲解)，在 Blender 右侧面板中找到 渲染属性 面板，勾选 环境光遮蔽 、 屏幕空间反射

因为摩托车大灯前面有玻璃材质，所以我们还需要打开 屏幕空间反射 里面的 折射，这样可以让玻璃产生折射效果，整体结果更真实

这时候，厉害可以调整 采样 > 视图 的数值，以获得更高的渲染精度

如果视图着色方式为「渲染」的话，这个操作将会显著提升 Blender 对硬件资源的占用，请酌情调整或把视图渲染方式调整为默认的「实体」

如果需要做出背景虚化，还可以点选摄像机，然后在它的 物体数据属性 面板中勾选 景深

接着展开 景深 的设置，可以看到，聚焦到物体 一栏目前是空值，这时候我们只需要点一些输入框后面的 吸取数据块 按钮，并在场景中点选摩托车，即可让摄像机聚焦到摩托车上

若有需要可以继续调整 焦点距离 参数

微调已经全部完成，此时我们可以在 Blender 上方菜单栏找到 渲染 > 渲染图像(或使用快捷键 F12)，这时就会弹出 Blender 渲染 窗口并渲染出最终效果

如果需要保存渲染结果为图片，我们可以在 Blender 渲染 窗口的菜单栏中找到 图像 > 保存 (or 另存为) 并根据自己的需要，选择输出格式、保存位置等参数，最后修改文件名并点击 保存为图像 按钮即可

接下来简单讲解如何输出动画

要输出动画，首先肯定要让画面动起来，这里演示最简单的动画效果，即物体不动，把摄像机移动到另一个位置

首先，确保 Blender 窗口下方的 时间轴 中，当前光标位于第0帧

接下来，我们需要点选摄像机，然后找到 Blender 右侧的 物体属性 面板

然后，把光标移动到物体属性面板的 变换 > 位置 上(X Y Z 均可)，然后按下键盘的 I 键

这时我们可以看到，位置参数输入框变成了黄色，并且在其后面出现了一个 实心菱形，同时时间轴的第0帧也出现了一个黄色菱形

这就意味着，摄像机目前的 位置参数 已经记录到时间轴的 第0帧 上

接下来，我们继续在摄像机的 旋转 参数上按下 I 键，把摄像机的拍摄角度也记录到时间轴上称为关键帧信息(同理，使用 Alt + I 可以取消记录关键帧信息)

接下来，我们打开 Blender 右侧的 输出属性 面板，在 格式 > 帧率 这里，我们可以看到这段动画每秒的帧率，在本例中，我们做一个5秒的动画，所以结束的一帧应该是第120帧，我们可以通过改变 帧范围 > 结束点 来让动画结束在120帧

所以我们先在需要选中时间轴的第120帧

这时，重新回到摄像机的物体属性面板，我们可以发现，之前被记录过关键帧的参数目前被绿色填充

为了更方便地进行操作，我们这里打开摄像机的 锁定摄像机到视图方位 并轻微调整观察角度

此时，我们再观察摄像机的位置属性，可以发现，由于摄像机改变了位置，所以位置和旋转参数都变成了新的值，数值框也变成了橙色

最后，我们重复前面记录关键帧的操作，把光标放在已改变的值的上面，按下 I 键，在第120帧的位置记录新的参数

这时，我们重新选择时间轴上的第0帧，并按下时间轴上方的播放按钮，就可以看到动画效果了

Blender 默认的插帧方式带有 「缓入缓出」 效果，我们可以通过先用鼠标框选时间轴上的关键帧(在Blender 3.5.1 中似乎不选中关键帧也行)，再在时间轴上点击鼠标右键，并在弹出菜单中找到 插值方式 > 线性 来关闭缓入缓出效果

接下来回到 输出属性 面板，根据需要更改 格式 > 分辨率，在该面板的下方还有 输出 > 输出路径 、文件格式 等参数，根据自身需要调整对应的项目即可

接下来找到 Blender 上方菜单栏的 渲染 > 渲染视频(或使用快捷键 Ctrl + F12)，让 Blender 开始渲染成品视频文件

渲染结束后，我们就可以在之前设定好的输出路径下找到输出的视频啦
