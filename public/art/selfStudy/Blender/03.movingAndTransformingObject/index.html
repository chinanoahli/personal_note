03. 变换和移动物体 #

移动物体 #

使用 Blender UI 上的按钮 #

在 Blender 视图左侧找到并点击进入 移动 模式，此时再点击任何物体，Blender 将会以物体的「原点」为基准，显示一个可操作的三轴操作杆(操作杆的末端为箭头状)，这时我们只需要用鼠标按住其中一个轴并在其轴向上拖移，即可在此轴上移动物体

细心的小伙伴可能会注意到，在任意 两条轴的中间 ，还会有一个带颜色的小方格

只需要拖动这个小方格就可以在 这两条轴所形成的平面上 任意移动物体，而不再受单一轴向的影响，注意观看下图方块的三轴座标变化

通过快捷键移动 #

在用鼠标点选物体之后，再按下键盘的 G 键，物体便会跟随鼠标的光标进行移动，当移动到合适的位置之后，按下鼠标的左键，即可确认物体当前的位置，若要中途推出移动物体的操作，只需要按下键盘的 Esc 键或 鼠标右键 即可

让物体在指定轴向上移动 #

如果需要让物体仅能在 X 轴上进行移动，则可以使用下列的快捷键操作:

  1. 按下键盘的 G 键，进入物体移动模式，此时物体可以随着光标任意移动
  2. 因为是需要把物体限定在 X 轴上移动，所以此时我们继续按下键盘的 X 键

这时候再移动光标，我们就能发现，物体仅能在 X 轴上改变位置

(其他轴也是同理，需要锁定在 Y 轴，则按 Y)

让物体在某两轴组成的平面上移动 #

假设我们先在需要把物体锁定在 X 和 Y 轴所组成的平面上(也就是说 Z 轴座标不变)进行移动的话，我们则可以通过:

  1. 按下键盘的 G 键，进入物体移动模式
  2. 按下快捷键 Shift + Z 来锁定物体的 Z 轴座标

这时候再移动光标，我们就能发现，物体仅能在 X 和 Y 轴所组成的平面上改变位置

让物体回到场景原点 #

如果我们需要把物体重新移动到场景的原点，则可以通过 Alt + G 来操作

通过快捷键精确移动 #

我们先在已经知道，让物体锁定在 Z 轴移动的快捷键是

我们都知道，可以通过 G → Y 快捷键来实现，那么有没有的更精确的 Key Combo 来让物体精 **「锁定 Y 轴并移动一定的位置」**呢？

答案当然是肯定的:

假设我们需要让物体锁定 Y 轴并往座标轴的负方向移动 2.5m 的距离，我们只需要按下快捷键 G → Y → - → 2 → . → 5 *，即「GY-2.5」，就可以让物体实现更精准的移动啦

*. 数字和符号可以直接用字母区上面的横排数字键和连字符进行输入，并不强制要求使用小键盘

变换物体 #

旋转物体 #

在 Blender 视图左侧找到并点击进入 旋转 模式，此时再点击任何物体，Blender 将会以物体的「原点」为基准，在物体的三个轴向平面上显示一道圆弧，此时我们只需要用鼠标按住其中一条圆弧，并往需要旋转的方向拖动，即可让物体在对应的轴向平面上作出旋转

当鼠标不选中任意一条圆弧时，物体则可以向任意角度旋转

通过快捷键旋转 #

旋转物体的快捷键是 R，与之前的操作类似，先选中物体，然后按下键盘的 **R**键即可

值得注意的是: 当第一次通过快捷键**R** 进入旋转模式时，物体仅能在与观察者视角的焦平面平行的方向上进行旋转(此时的光标会变成一个两向的黑色箭头)，效果如下图

如果我们需要物体进行任意旋转模式，则需要连续使用 R 快捷键(此时的光标会成一个红绿垂直的十字箭头样式)

在指定的轴向上旋转 #

与移动物体类似，如果我们需要让物体以 X 轴为中心旋转的话，只需要按下快捷键 R → X 再移动光标即可

精准角度控制则可以使用 R → X → 45 这种形式的 Key Combo

旋转归零 #

同样地，与移动物体类似，你可以通过快捷键 Alt + R 快速地将一个旋转过的物体重置为 0°，

缩放物体 #

在 Blender 视图左侧找到并点击进入 缩放 模式，此时再点击任何物体，Blender 将会以「原点」为基准，显示一个可操作的三轴操作杆(操作杆的末端为一个小方块)，这时我们只需要拖动这个小方块，就可以让物体在对应的轴向上拉长或缩短

同时，与移动物体类似，在操作杆任意两轴组成的平面上也有一个小方格，可供用户在两条轴上同时缩放物体，而不影响第三条轴

通过快捷键缩放 #

缩放的快捷键是 S，默认的缩放模式是 「等比缩放」

与移动和旋转物体类似:

你也可以通过 S → X 这种形式来限定物体，让其仅能在 X 轴向上进行缩放

而 S → Shift + Z 这种限定方式，则可以让物体 锁定 在 X 和 Y 轴所组成的平面方向上进行 「等比缩放」

缩放归零则可以使用 Alt + S 快捷键

同时移动与变换物体 #

使用 Blender UI 上的按钮 #

Blender 还提供了一个 变换 模式，激活后，视图中可以同时显示物体的 移动操作杆 、 旋转操作弧 以及 缩放操作杆，此时用户只需要根据自己的需要对物体作出调整即可

Gizemo #

诚然，Blender 在 UI 设计上给予了初学者很大的便利，但通过鼠标选择工具才能进入对应的操作模式这种工作方式是比较低效的

那么有什么办法可以让我们在点选一个物体的时候，能更快显示出常用的操作杆呢？

那就是 Gizemo 啦，只要在视图右上角的 Gizemo 设置菜单里，根据需求勾选位于 物体Gizemo 类别下的: 移动 、 旋转 或 缩放，那么我们在下次选点选物体时，对应的操作杆将会自动显示出来

通过快捷键 Ctrl + ` 或直接点击视图上的 显示Gizemo 按钮则可以快速开启或关闭便利的 Gizemo 功能

通过参数面板 #

如果你有更数学精确上的要求，那么你可以使用 Blender 提供的物体参数面板，或通过快捷键 N 激活的 N 键面板 来手动输入物体的座标、旋转角度和变形参数
