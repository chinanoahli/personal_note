乐理学习笔记 #

⚠ 本笔记基本上是基于 三分钟音乐社@Bilibili 的 视频 BV14p4y1e7TV 进行学习

💡 大部分钢琴示意图都是 SVG 矢量格式，如果看不清，你可以右键点击图片，并在新标签页中打开，然后就可以任意缩小放大

⚠ 部分 SVG 会在打开了 强制黑暗模式 / Auto Dark Mode for Web Contents 的Chorme 浏览器中出现色彩错误的现象，你可以暂时关闭该模式，或使用其他浏览器进行浏览

💡 乐理的学习通常以钢琴作为演示，原因是因为钢琴的结构更简单更直观，可以更容易地解释乐理知识

乐理通用于所有乐器，所以并不会出现在学习乐理时使用钢琴，学完后却不会使用其他乐器的情况

⚠ 本篇笔记在大多数情况下都使用了 科学音高记号法 来标识音名

需要先行解释的词汇概念 #

音高 #

音高是指人类心理上对音符基本频率的感受，不同的音符（可以先简单理解为钢琴琴键）有不同的音高

音名 #

音名，即音高的名字，为了方便音乐交流，通常以字母 C D E F G A B 来表示

音域 #

音域是指特定一种乐器（包括人声），能够演奏出来的 所有 的 音高 的总和

钢琴的音域为 A ~ C8 （音乐符号的解释详见后文）

未经专业训练的大部分人声，音域通常为 G3 ~ D5

简谱与唱名 #

简谱是音乐的记谱法之一，主要以数字作为记号来表示音乐，绝大多数的简谱看起来都跟下图很像，下图为《奇异恩典》的简谱

标题 #

与绝大多数的其他文献一样，图中一开始的粗体居中字为简谱的标题 AMAZING GRACE

调号 #

接下来出现的是 调号 1 = C，调号通常会出现在标题下方的第一行开头

1 = C 的意思即为让唱名 do ，与音高 C 对应

1 = C 这种形式是最常见的，但也会出现 1 = D 、1 = E 或是 2 = C 、 3 = C 等情况

唱名 以及 C 的解释将会于后文中出现，届时将会出现更多关于调号的补充

拍号 #

紧跟在 1 = C 之后出现的是一个类似数学上分数的符号 3 / 4，这个就是拍号

拍号决定了这份乐谱的节奏（律动）

后文会有更详细跟解释，音拍号还包含 音符 、 时值 、 节奏 、 速率 、 小节 、 小节线 等等概念有关联之处，这里就先不展开说明

音符与唱名 #

接下来就是乐谱主体的部分 —— 音符

我们可以观察到，简谱的音符是以数字表示的，分别用的是数字 0 ~ 7

每个音符都有它们独特的名字，它们的名字统称为 唱名

音符 0 1 2 3 4 5 6 7
唱名 do re mi fa sol la si

0 为 休止符 ，代表暂停演奏和演唱，故不存在唱名

⚠ 音符的 数字表示 与唱名 永远是 一一对应 的

那该如何在简谱上表现高低音呢？

用 高 / 低音点 即可

要提高的音只需要在数字上方加点圆点，要提高 1 次，就加点 1 个圆点；要提高 2 次，则加点两个圆点，最高不超过 4 个原点

要降低的音只需要在数字下方加点原点，同理，按照降低的次数加点圆点，且最高不超过 4 个圆点

其他相对没那么重要的标记 #

情绪 #

还有一些乐谱会在拍号的后面用文字描述出 情绪 ，这可以让唱谱的人更直观地快速把握到乐谱的整体感觉

会用于语情绪的用词有：欢快地 、 激昂地 、 悲伤地 、 令人振奋地 等等…

速度 #

有些乐谱会在情绪后面用符号或文字描述出 速度 ，如：

  • 较为精确的写法 ♩ = 130
  • 较为抽象的写法 快速

同样地，速度 的作用是为了给予唱谱人便利

实际上 情绪 和 速度 通常都能被 拍号 更为精准地描述，所以乐谱上不论是否出现情绪与速度，实际上影响都没有十分大

更多的解释同样后文会有详细的解释

认识钢琴 #

琴键的名称 #

长长的钢琴由 9 个完全一致的 组 构成

每个组包含了 7 个白键和 5 个黑键，而且左边的按键发出的音高总比右边的按键发出的音高要低

⚠ 注意: 一般的钢琴总共有 88 个按键，虽然最左边和最右边的两组都 不包含 完整的 12 个按键，但它们仍然算是完全独立的两个组

⚠ 注意: 虽然钢琴的白键和黑键在外观上有高有低，但它们的地位是完全相等的，这种有高有低的设计是为了可以更轻松地演奏而设计的

⚠ 注意: 虽然钢琴上只有 9 个组，但实际上音的范围并不只有这 9 组的范围，音的范围是无穷无尽的，往钢琴两端无限延伸

每一个组的 7 个 白键 正好对应 7 个音名，从左到右依次分别都是 C D E F G A B

除了白键，我们可以看到每一组按键中还有 5 个黑键，那么黑键又要如何称呼呢？

变音记号和「等音」 #

在了解黑键的名称之前，我们还需要先了解一下音乐中的 变音记号: 升号 ♯ 和 降号 ♭

升号代表要从相邻的音提升，降号则代表要从相邻的音降低，带升降号的音名读作 升某音 或 降某音

第 1 个黑键位于 C 之后，代表要从 C 提升，因此记作 C♯ 读作: 升 C

同时它也在 D 的前面，所以它也能代表要从 D 降低，因此也可以记作 D♭ 读作: 降 D

以此方法进行推理就可以知道，第 2 个黑键它位于 D 之后和 E 之前，所以它也可以记作 D♯ 或 E♭

此时，我们就称 D♯ 和 E♭ 是 等音，等音 正是指同一个音的不同名称

因此，白键与黑键的名称应该如下图显示的一般

两种不同的音高记号法 #

前面我们说过，钢琴是由 9 个完全相同的组所组成，而每个组的键的名称又完全一致，所以必定存在一种方法可以区分音名相同且位于不同的组的按键

在国际上流行两种 音高记号法 ，它们分别是 科学音高记号法 和 赫尔姆霍茨音高记号法

科学音高记号法 #

科学音高记号法 (Scientific pitch notation, SPN) 又被称为 美国标准音高记号法 (American standard pitch notation, ASPN) 或 国际音高记号法 (International pitch notation, IPN)

在 科学音高记号法 中，全部琴键的音名都用大写字母标记

钢琴最左侧不完整的一组为第 0 组，从第 0 组算起，从左到右依次递增

第 0 组的音名不需要标注组序号，直接记作 A 、 B

从第 1 组开始，则需要标注组序号，记作 C1 、D1 、 C2 、 D2 等，黑键则根据白键的音名作同样的变形即可

下图为 科学音高记号法 的钢琴全键名称示意图

亥姆霍兹音高记号法 #

亥姆霍兹音高记号法 是由德国科学家 赫尔曼·路德维希·费迪南德·冯·亥姆霍兹 (Hermann Ludwig Ferdinand von Helmholtz) 所发明的一套音高记号法

通常国内又称之为 赫尔姆霍茨音高记号法

在 亥姆霍兹音高记号法 中，从左数起第三组被称为 大字组，第 1 组的组名 不包含 序号，大字组的序号往 左 递增

即：大字组、大字一组、大字二组

大字组 的音名要用 大写字母 表示，并将组的序号写在 右下角 ，这里列出几个实例

键的位置 音名
大字组的 C C
大字一组的 C C1
大字二组的 C C2

大字组 的的右边一组被称为 小字组，同样地，第 1 组小字组也 不包含 序号，小字组的序号往 右 递增

即：小字组、小字一组、小字二组

小字组的音名要用 小写字母 表示，并将组的序号写在 右上角 ，这里列出几个实例

键的位置 音名
小字组的 C c
小字一组的 C c1
小字二组的 C c2

同样地，其他键的音名根据这种模式推出来即可，下图为 亥姆霍兹音高记号法 的钢琴全键名称示意图

💡 原版的 亥姆霍兹音高记号法 是以撇号来标记音名的:

大字X组 的 C 需要在 C 的右下角加上 X 个 ͵ ，即 C͵ 表示 大字一组 的 C ； C͵͵ 表示 大字二组 的 C

小字X组 的 c 则需要在 c 的右上角加上 X 个 ′ ，即 c′ 表示 小字一组 的 c ； c′′ 表示 小字二组 的 c

后来人们发现这种方法比较难以辨识，就将撇号改变成数字的样式

变音记号的补充 #

𝄪 重升号 (Double Sharp) #

从某个音高提升 两次 ，可以用 重升号 𝄪 来表示，如: 重升 C 可以表示为 C𝄪

𝄫 重降号 (Double Flat) #

从某个音高降低 两次 ，可以用 重降号 𝄫 来表示，如: 重降 C 可以表示为 C𝄫

因此除了 G♯ / A♭ 以外，所有的琴键都有三个等价的名称

1 = C 的补充 #

在对唱名、钢琴的结构、音名、和音高记号法之后，这里需要对前面简谱 1 = C 作一些内容补充

绝大部分的简谱上的 1 = C ，是指让唱名 do 对应到 C4 音高，原因主要有两点：

  1. 前面我们提到过：简谱绝大部分都是为了写给音乐入门新人以及音域在一般人范围的歌者使用，他们的音域绝大部分都落在 G3 ~ D5 的范围，为了让读谱的人更方便，自然是需要把最常唱出的音对应在大部分人的音域范围中
  2. 简谱的高低音点如果太多，会让简谱变得很复杂，为了尽可能与钢琴对应，将 do 对应到 C4 ，就可以尽可能减少高低音点的使用，从而达到简化谱面的效果

中央 C #

中央 C 是指钢琴上的 C4 ，也就是 1 = C 通常所对应的 C

标准音 #

标准音 是指 A4 ，它的频率国际上公认地固定为 440 Hz ，而钢琴上其他按键的频率，都将它作为参照而进行校准

💡 标准音 的频率不是永远固定不变的，国际上每过一段时间都会对其进行一次调整

纯八度 #

本内容为 音程 相关知识的子项目，后面说道 音程 时仍会详细介绍，在此先作简单的介绍

当两个音的频率数值比为 1 : 2 时，我们成它们为 纯八度

在钢琴 相邻 的任意两个组之间，音名相同的键所发出的音 (如: C3 和 C4 ，D2 和 D3 ) ，都为 纯八度 关系，如果两个相音名相同的音相隔多组，我们可以称之为 N个纯八度 的关系

互为 纯八度 的音听起来极为相似、和谐，同时弹奏的话，对音不敏感的人听起来甚至会像是只有一个音一样

这也是为什么琴键的音名只有 7 个，因为到第 8 个时，它发音的频率正好是第 1 个的两倍，它们听起来极为相似，听起来几乎只有一个音，所以就不再使用新的音名了

全音与半音 #

C4 D4 E4 ，三个键的频率分别为 261.63 Hz 293.66 Hz 329.63 Hz ……

如果音乐的世界都需要用赫兹来沟通的话，将会变得非常麻烦

所以为了简化音乐的沟通，人们人为地将两个相邻音名的差值简称为 半音

半音 是形容两个相邻的音 (如: B 和 C ， C 和 C♯ ) 之间的 距离 (距离是不分方向的，既可以往低音的方向找，也可以往高音的方向找) 的词汇，当只有一个音的时候，是不可以使用 半音 这个词汇的，因为并没有对比的对象

而如果两个连续 两个 半音 的距离加起来 (如: B 和 C♯ ， C 和 D) ，就是一个 全音

所以 半音 和 全音 的准确说法应该是：

  • B 和 C 是一个 半音 的距离
  • B 和 C♯ 是一个 全音 的距离

但是在日常的交流中，我们通常会简化成以下读说法：

  • B 和 C 是 半音
  • B 和 C♯ 是 全音

同时，在任何乐器中，B 和 C 、 E 和F 都是直接相连在一起的

1~7̇1 的内在规则 #

实际上，这里研究的只有 do ~ si 的内容，不过为了不忽略前一组与后一组的关联，所以也会加上后一组的 do 进行讲解

音乐里有一项很奇妙的现象， do ~ si do 音列听起来永远都都是 do ~ si do 而不会跟其他音列混淆

我们都知道当 1 = C 的时候，1~7̇1 正好都对应在白键 C D E F G A B C 上，

先在来分析一下这些白键的之间的关系:

就可以发现， do ~ si do 音列实际上对应的是一个 全 全 半 全 全 全 半 这样一个音的差距之间的循环组合

也就是说我们在弹奏 do ~ si do 时，必须完全复现这一规律，才能让 do ~ si do 听起来像是 do ~ si do

如果我们让 1 = D 的话，按照 全 全 半 全 全 全 半 这组序列推算，就必须要弹奏 D E F♯ G A B C♯ D ，如下图一样

同理让 1 = E 的话，音列就会变成 E F♯ G♯ A B C♯ D♯ E

这个 全 全 半 全 全 全 半 套用在黑键上也是完全一致的，只要按照这个序列推算出来的音列，一定都会给人有 do ~ si do 的感觉

那么，这个 全 全 半 全 全 全 半 到底是什么东西呢？

调式 #

对于 调式 的定义，简单来说就是: 把 若干个 音，按照 某种规则 排列起来

由于音乐中，这个 某种规则 里有很多种不同样的规则，这里就先单独介绍 全 全 半 全 全 全 半 这一种

💡 调式 里面还包括: 大调式 、 中古调式 、 小调式 、 五声调式 等等调式，这里就不一一列举

不同的 调式 所使用到的音的数量以及排列规则都不一样

自然大调 #

💡 自然大调 ，全称为 自然大调式

但在日常交流中，人们通常会把 大调式 中的「式」字省略

对于 自然大调 人们还通常还会省略「自然」二字，直接称呼其为 大调

自然大调式 只是 大调式 的其中一种调式

自然大调 就是指 7 个音，按照 全 全 半 全 全 全 半 的规则排列起来

你可能会问，被 7 个 全、半音 分离开的音不是应该有 8 个才对吗？

没错，确实被 7 个 全、半音 分离开的音是 8 个

但仔细分析过后，你会发现: 最后一个音只是第一个音的高八度，音名是一样的

正如上面提到的 do ~ si do 一样，最后一个音 do 正是一开始的 do 的高八度，这就说明这个 调式 已经结束并回到最开始的第一个音

所以这里我们说 大调 里面只有 7 个音

既然 大调 只规定了音的数量和排列规则，那么我们直到 1 即可以 = C 也可以 = D, E, F 等等，我们要怎么称呼这些从不同音开头的 大调 呢？

答案很简单，我们只需要在「大调」的前面加上 音名 即可，如:

C 大调 、 D 大调 等等……

音阶 #

音阶 的定义为: 将 调式 中的音，以 主音 开始 到 主音 结束 ，由 从低到高 (叫做: 上行)，或 从高到低 (叫做: 下行) ，以 阶梯状 排列起来
